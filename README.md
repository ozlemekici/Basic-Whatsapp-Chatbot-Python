# BasicWhatsappChatbot
Basic Whatsapp Chatbot by using Python

This work has started in response to the question "How to send automatic messages using python with Whatsapp web". Accordingly, a bot was built in a whatsapp group that automatically replies to a specific message.

You will not be able to use this program directly by copying the final version. (Due to variables such as xpath, coordinate)
Installations are valid for the Windows operating system. The explanation has been made over PyCharm and it is based on the Chrome driver.

It was written assuming basic knowledge of python.

With "Selenium", we will see automatic data entry to a site and data retrieval from a site (Xpath), using a data copied with CTRL + C on a computer in python (pyperclip), automatic mouse and keyboard routing using python (pyautogui).
